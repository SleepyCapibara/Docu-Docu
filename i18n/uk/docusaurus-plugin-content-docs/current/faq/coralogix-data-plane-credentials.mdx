# Як отримати облікові дані для Coralogix Data Plane

<DocAuthors authors={["Sergey Bayrachny"]} date="5 вересня 2025 р." />

1. Виберіть значення у випадаючому списку **Region** у налаштуваннях Data Plane на платформі SOC Prime. Ви можете визначити його за URL-адресою розгортання Coralogix: знайдіть ім’я хоста вашої команди у вашій URL-адресі та зіставте його з регіоном AWS у таблиці Domains [тут](https://coralogix.com/docs/user-guides/account-management/account-settings/coralogix-domain/#domains).
2. Створіть API-ключ Coralogix.
   1. На панелі інструментів Coralogix натисніть значок користувача у верхньому правому куті та натисніть **Settings**.
   2. У лівому меню виберіть **Keys**.
   3. Натисніть **+ TEAM KEY.**
   4. Додайте назву ключа.
   5. Надайте ключу дозвіл Alerts за допомогою пресетів ролей.
   6. Натисніть **Create**.
   7. Натисніть Download, щоб зберегти деталі ключа у форматі JSON.

![Знімок екрана продукту](/img/docs/faq/coralogix-data-plane-credentials/01.png)

3. Налаштуйте параметри у вашому Data Plane на платформі SOC Prime.
4. Відкрийте завантажений JSON-файл із деталями ключа:
   1. Скопіюйте значення ключа **keyId** і вставте його у поле **API Key ID** у налаштуваннях Data Plane на платформі SOC Prime
   2. Скопіюйте значення ключа **keyValue** і вставте його у поле **API Key** у налаштуваннях Data Plane на платформі SOC Prime
