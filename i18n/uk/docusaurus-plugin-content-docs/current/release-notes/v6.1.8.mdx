## Оновлення Active Threats

Ми покращили модуль Active Threats, удосконаливши сортування детекцій. При відкритті сторінки деталей Active Threat і переході на вкладку Detections детекції у всіх підвкладках (Behavior Rules, IOC Queries і AI Rules) тепер сортуються за датою, причому найновіші елементи відображаються зверху.

![Знімок екрана продукту](/img/docs/release-notes/v6.1.8/12.png)

---

У найновішому випуску ми покращили візуальне оформлення PDF-звітів Scan і Data Audit у Attack Detective, щоб їх було легше читати, навігувати та виявляти критичні знахідки.

- Покращено функціональність сортування в PDF-звітах Scan, щоб легше визначати найкритичніші та найактивніші детекції. Детекції тепер сортуються за рівнем серйозності (Critical → Low), а в межах кожного рівня серйозності — за кількістю спрацювань, від найвищої до найнижчої.
- Для покращення читабельності ми додали візуальні роздільники в PDF-звіти Data Audit, що полегшує розрізнення подій.

---

Ми впровадили кілька покращень для забезпечення точніших перекладів на певних платформах.

## Elastic

Для покращення якості перекладів у формат Elastic Stack Detection Rule (Lucene) ми забезпечили, що поле `reference` порожнє, коли немає `case id`.

## Crowdstrike

Ми усунули проблему перекладу ключового слова Crowdstrike, взявши значення `keyword` у подвійні лапки (`"`).

## Tanium

Ми покращили якість перекладів Tanium Query, удосконаливши екранування `\\`.

---

- Виправлено проблему в режимі New, коли контент у редакторі іноді не розпізнавався під час виконання AI-завдань після переходу з вкладки Detection Code на сторінці Detection Rule.
- Усунуто проблему, коли натискання кнопки Revoke в Uncoder AI іноді некоректно викликало спливаюче вікно Publish this content? замість повідомлення Rule successfully revoked.

- Покращено формулювання та узгодженість тексту на сторінках Attack Detective Scan Setup і Two-Factor Authentication Setup.
- Покращено поведінку з’єднувальних ліній у Attack Flow шляхом коригування алгоритму розміщення ліній.
- Виправлено проблему в Uncoder AI, коли в деяких випадках бічне меню було частково обрізане під час перегляду результатів завдання Active Threats Search.
- Виправлено проблему в Advanced Search, коли кнопка Translate To іноді не відображалася при виборі детекцій для масових дій.
- Усунуто проблему, коли підкреслення вкладок Generate, Translate і Improve в Uncoder AI іноді відображалися некоректно.
- Скориговано рівень масштабування для малого Relation Graph на сторінці Detection Rule для покращення видимості.
- Виправлено проблему, коли опція Unlock all Premium rules відображалася за межами випадаючого списку на сторінці Lists.
- Заблоковано кнопки дій у меню з трьох крапок для DEMO Audits у Attack Detective та виправлено проблему, коли підказка для цих дій іноді розміщувалася некоректно.
