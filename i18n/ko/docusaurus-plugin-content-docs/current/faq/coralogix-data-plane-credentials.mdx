# Coralogix Data Plane 자격 증명을 가져오는 방법

<DocAuthors authors={["Sergey Bayrachny"]} date="2025년 9월 5일" />

1. SOC Prime Platform의 Data Plane 설정에서 **Region** 드롭다운의 값을 선택합니다. Coralogix 배포 URL로 Region을 확인할 수 있습니다. URL에서 팀 호스트 이름을 찾아 Domains 표의 AWS Region과 일치시키세요. [여기](https://coralogix.com/docs/user-guides/account-management/account-settings/coralogix-domain/#domains)에서 확인할 수 있습니다.
2. Coralogix API 키를 생성합니다.
   1. Coralogix 도구 모음에서 오른쪽 상단의 사용자 아이콘을 클릭한 다음 **Settings**를 클릭합니다.
   2. 왼쪽 메뉴에서 **Keys**를 선택합니다.
   3. **+ TEAM KEY.**를 클릭합니다.
   4. 키 이름을 추가합니다.
   5. 역할 프리셋을 사용하여 키에 Alerts 권한을 부여합니다.
   6. **Create**를 클릭합니다.
   7. Download를 클릭하여 키 세부 정보를 JSON으로 저장합니다.

![제품 스크린샷](/img/docs/faq/coralogix-data-plane-credentials/01.png)

3. SOC Prime Platform에서 Data Plane 매개변수를 구성합니다.
4. 다운로드한 키 세부 정보 JSON 파일을 엽니다.
   1. **keyId** 키의 값을 복사하여 SOC Prime Platform의 Data Plane 설정 **API Key ID** 필드에 붙여넣습니다.
   2. **keyValue** 키의 값을 복사하여 SOC Prime Platform의 Data Plane 설정 **API Key** 필드에 붙여넣습니다.
