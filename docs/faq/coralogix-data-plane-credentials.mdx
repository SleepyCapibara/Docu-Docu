# How to Get Credentials for Coralogix Data Plane

<DocAuthors authors={["Sergey Bayrachny"]} date="September 5, 2025" />

1. Select the value in the **Region** dropdown in the Data Plane settings on the SOC Prime Platform. You can determine it by your Coralogix deployment URL: find your team's host name in your URL and match it to an AWS Region in the Domains table [here](https://coralogix.com/docs/user-guides/account-management/account-settings/coralogix-domain/#domains).
2. Create the Coralogix API key.
   1. From the Coralogix toolbar, click the user icon at the top right corner and click **Settings**.
   2. In the left-hand menu, select **Keys**.
   3. Click **+ TEAM KEY.**
   4. Add a key name.
   5. Attach Alerts permission to the key using role presets.
   6. Click **Create**.
   7. Click Download to save the key details as JSON.

![Product screenshot](/img/docs/faq/coralogix-data-plane-credentials/01.png)

3. Configure parameters in your Data Plane on the SOC Prime Platform.
4. Open the downloaded JSON file with key details:
   1. Copy the value of the **keyId** key and paste it into the **API Key ID** field in the Data Plane settings on the SOC Prime Platform
   2. Copy the value of the **keyValue** key and paste it into the **API Key** field in the Data Plane settings on the SOC Prime Platform
